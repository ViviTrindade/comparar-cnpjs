Diferencial

Neste projeto, utilizei Python para automatizar a comparação de bases de CNPJs. Para acelerar o desenvolvimento do script e esclarecer dúvidas durante o processo, contei com o suporte de uma IA, que me ajudou a estruturar o código e a lógica de comparação. Todo o trabalho prático de implementação, testes e ajustes foi feito por mim.
